# 验机侦探 macOS 上架指南

## Apple Developer 账号申请

### 步骤
1. 访问 https://developer.apple.com/programs/enroll/
2. 选择 "Individual" (个人) 或 "Organization" (公司)
3. 支付年费 $99 (个人) 或 $299 (公司)
4. 等待审核 (1-5个工作日)

## Flutter Desktop 应用签名配置

### 1. 配置 macOS Runner
在 `device_inspector/macos/Runner/` 目录下：

**Info.plist 必要配置：**
- CFBundleDisplayName: 验机侦探
- CFBundleIdentifier: com.lianggou.deviceinspector
- CFBundleVersion: 1
- CFBundleShortVersionString: 1.0.0
- LSMinimumSystemVersion: 10.15

### 2. 添加权限描述 (Entitlements)
创建 `macos/Runner/Runner.entitlements`:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>com.apple.security.app-sandbox</key>
    <true/>
    <key>com.apple.security.files.user-selected.read-write</key>
    <true/>
    <key>com.apple.security.device.usb</key>
    <true/>
</dict>
</plist>
```

### 3. macOS Catalyst 配置 (如需支持跨平台)
在 `pubspec.yaml` 添加：
```yaml
macos:
  bundleIdentifier: com.lianggou.deviceinspector
```

## 沙盒权限配置

### USB 设备访问
iOS设备通过USB连接需要：
- com.apple.security.device.usb entitlement
- 用户手动授权

### 网络访问
- App Sandbox 下需要 com.apple.security.network.client entitlement
- 用于通行证验证和报告上传

## App Store Connect 配置

### 1. 创建 App Store Connect 记录
1. 登录 https://appstoreconnect.apple.com
2. My Apps → + → New App
3. 填写：Platform: macOS, Name: 验机侦探

### 2. 上传构建版本
```bash
# 构建 release 版本
flutter build macos --release

# 使用 Transporter 上传
# 或 Xcode 直接发布
```

### 3. 填写 App 信息
- 隐私政策 URL (必须)
- 分类: Utilities
- 年龄分级: 4+

## 常见拒绝原因与解决方案

| 原因 | 解决方案 |
|------|---------|
| 沙盒权限不足 | 添加必要的 entitlement |
| 隐私描述缺失 | 在 Info.plist 中添加 NSBluetooth/NSCamera 等描述 |
| 网络超时 | 增加超时配置 |
| 账号信息不完整 | 完善 App Store Connect 账号信息 |
